# Vision

Build one clean open-source platform blueprint, deploy it separately per instance, and run an IT freelancer company through approval-gated agent workflows.

Company operating model:
- 1 human CEO as final approver for high-risk decisions.
- Personal assistant capability for inbox and calendar triage with a prioritized daily list.
- CTO/CIO agent capability for customer-facing technology counsel and continuous internal platform improvement.
- Chief AI / Digital Strategy capability for AI engineering, digitalization, and data engineering advisory/delivery.
- Agents cover corporate functions (billing, accounting, CFO strategy, finance, reporting, PO, compliance).
- Agents cover service delivery functions (PM, delivery, quality, consulting, documentation, testing).
- Mission Control UI shows agent status, workflow state, approvals, and audit history.
